# 🛒 MERN E-Commerce Platform

A full-stack E-Commerce application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). It provides a seamless shopping experience with secure authentication and product management.

## ✨ Features
- User Registration & Login (JWT Authentication)
- Browse & Search Products
- Add to Cart
- Place Orders
- Admin Dashboard for Product Management
- Responsive UI

##  Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- JWT
